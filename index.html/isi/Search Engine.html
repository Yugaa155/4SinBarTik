2. Search Engine

A. Definisi

1. Web Browser adalah aplikasi atau software untuk membuka dan menampilkan halaman web dari internet.

2. Search Engine adalah layanan untuk mencari informasi di internet menggunakan kata kunci.

3. Variabel Mesin Pencari adalah pengaturan atau trik untuk menyaring hasil pencarian.

4. Pocket adalah aplikasi untuk menyimpan artikel atau halaman web agar bisa dibaca secara offline.

B. Penjelasan

Fungsi Web Browser: Menampilkan konten web, navigasi antar halaman, pencarian informasi, bookmark, dan sebagainya.

Contoh: Google Chrome, Mozilla Firefox, Microsoft Edge, Safari, Opera.

Fungsi Search Engine: Memberikan informasi melalui proses crawling, indexing, dan ranking.

Contoh: Google, Bing, Yahoo, Baidu, DuckDuckGo, Yandex.

Variabel Pencarian: Kata kunci, tanda kutip, operator boolean, site:, waktu, filetype:, lokasi, topik, pertanyaan.

Pocket: Archive dan Favorites. Cara pakai: klik ikon, masukkan URL, klik “Save”.

C. Kesimpulan

Web browser digunakan untuk menjelajah internet, search engine untuk mencari informasi.

Mesin pencari bekerja dengan crawling, indexing, dan ranking.

Variabel pencarian membantu pencarian yang lebih akurat.

Pocket menyimpan konten untuk dibaca kembali.
